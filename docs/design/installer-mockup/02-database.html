MySQL の接続情報を入力してください。
この情報は契約中のレンタルサーバーの管理画面（コントロールパネル）の「データベース」欄で確認できます。

最大の詰まりどころ: 共有ホスティングのホスト名は localhost とは限らない（例: ヘテムル mysqlXXX.heteml.lib / さくら mysqlXXX.db.sakura.ne.jp）。プレースホルダ・ツールチップ・「ホストの調べ方」リンクで強く誘導したい。

初期値は localhost。共有ホスティングでは専用ホスト名に書き換えることが多いです。

通常はそのまま（3306）。

事前に作成した空のデータベースを指定します。テーブルはこのインストーラが作成します。

サーバーの DB ユーザーのパスワード。NeNe Invoice のログインパスワードとは別物です。

ローディング状態が必須: このボタンは「接続確認 → テーブル作成（スキーマ適用）」を行い数秒かかりうる。押下後はボタンを「適用中…」に変えてスピナー表示＋二重送信防止。失敗時は同画面にエラー表示で入力値を保持。
成功時: ステップ2（管理者設定）へ遷移。失敗時: 「DB 接続エラー: （詳細）」を上部バナーに赤表示（→ 状態・エラー集）。

出典: public_html/install.php step 1（db_host / db_port / db_name / db_user / db_pass）
